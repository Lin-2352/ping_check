# ⚡ Quick Start Guide - MC5 Ping Monitor

**Want to see your ping while playing Modern Combat 5? Here's the fastest way:**

---

## 🎮 Easiest Method (3 Steps)

### Step 1: Open the `new` folder
Navigate to: `new` folder in your ping_check directory

### Step 2: Double-click `StartPingOverlay.bat`
This starts the ping overlay (it will wait for MC5 to start)

### Step 3: Launch MC5 and play!
The ping overlay will automatically appear when your game window is visible

**That's it!** 🎉

---

## 📍 What You'll See

A small overlay in the top-left corner showing:
```
MC5 PING: 45 ms
```

**Colors:**
- 🔵 **Cyan (Blue)** = Good ping (<100ms)
- 🟡 **Yellow** = Fair ping (100-200ms)
- 🔴 **Red** = Poor ping (>200ms)

---

## 🎯 Pro Tips

**Move the overlay:** Click and drag it anywhere you want

**Close the overlay:** Right-click on it

**Auto-hide:** The overlay automatically hides when you minimize the game

**Auto-close:** The overlay closes when you exit MC5

---

## 🔄 Alternative: RTSS Integration

If you already use **MSI Afterburner** or **RivaTuner Statistics Server (RTSS)**:

1. Double-click `StartPingMonitor.bat` (in main folder)
2. Configure RTSS to display the ping file
3. See `USAGE_GUIDE.md` for detailed RTSS setup

## ❓ Problems?

**Overlay not showing?**
- Make sure MC5 is running
- Game window must be visible (not minimized)

**Shows "TIMEOUT"?**
- Check your internet connection
- MC5 servers might be down

**Need more help?**
- Read `USAGE_GUIDE.md` for detailed instructions
- Check `README.md` for troubleshooting
